# 🩺 AI-Powered Virtual Doctor

A Machine Learning-powered healthcare assistance system that analyzes user-provided symptoms and predicts possible medical conditions. The application provides confidence scores, recommendations, and precautionary measures through an interactive web interface built with Gradio.

## Features

* Symptom-based disease prediction
* Confidence score classification (High / Medium / Low)
* Disease-specific recommendations
* Precaution suggestions
* Emergency symptom detection
* Interactive and user-friendly web interface
* Input validation for reliable predictions

## Technologies Used

* Python
* Pandas
* Scikit-learn
* NLTK
* Gradio
* Machine Learning (TF-IDF + Naive Bayes)

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Powered-Virtual-Doctor.git
cd AI-Powered-Virtual-Doctor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

The application will launch locally and provide a web-based interface for symptom analysis.

## How It Works

1. Enter one or more symptoms.
2. Provide age information.
3. Select gender.
4. Submit the form.
5. The system analyzes the symptoms using a trained Machine Learning model.
6. The predicted condition, confidence score, recommendations, and precautions are displayed.

## Sample Input

```text
fever, cold, cough
```
